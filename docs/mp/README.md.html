README.md¶

 1# IBM&reg; Decision Optimization Modeling for Python (DOcplex)
 2
 3Welcome to the IBM&reg; Decision Optimization Modeling for Python.
 4Licensed under the Apache License v2.0.
 5
 6With this library, you can quickly and easily add the power of optimization to
 7your application. You need IBM ILOG CPLEX Optimization Studio to solve the models.
 8
 9This library is composed of 2 modules:
10
11* IBM&reg; Decision Optimization CPLEX Optimizer Modeling for Python - with namespace docplex.mp
12* IBM&reg; Decision Optimization CP Optimizer Modeling for Python - with namespace docplex.cp
13
14Solving with CPLEX requires that IBM&reg; ILOG CPLEX Optimization Studio V12.10 or later
15is installed on your machine.
16
17This library is numpy friendly.
18
19## Install the library
20
21```
22   pip install docplex
23```
24
